Docker

This project can optionally be run and managed as a Docker container.

Build the Docker image
^^^^^^^^^^^^^^^^^^^^^^

docker build . -t  web-youtube-dl:latest --force-rm

Run the service
^^^^^^^^^^^^^^^

docker-compose up -d
